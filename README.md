# Pong Game

This is a basic pong game that used HTML5 Canvas. The purpose of this project is to explore the ES6 by using Object Oriented Programming (OOP).

#Technologies Used

* HTML5
* CSS3
* JavaScript ES6
* webpack


## Keys

##### Player 1:
* a: up
* z: down

##### Player 2:
* ▲ : up
* ▼: down
